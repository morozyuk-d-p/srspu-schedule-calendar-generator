Расписание ЮРГПУ НПИ: конвертер в формат iCalendar

Описание

Это микроприложение для динамической конвертации расписания ЮРГПУ НПИ в формат iCalendar, пригодный для использования с большинством приложений-календарей.

Как использовать?

URI-схема запроса для создания календаря расписания занятий:
/generate/ics/classes?faculty=foo&year=bar&group=spam
Здесь foo - код факультета (смотрите таблицу ниже), bar - курсы с 1 по 6, spam - краткий шифр группы (к примеру, ПИа).

URI-схема запроса для создания календаря расписания зачётов:
/generate/ics/finals?faculty=foo&year=bar&group=spam
Здесь foo - код факультета (смотрите таблицу ниже), bar - курсы с 1 по 6, spam - краткий шифр группы (к примеру, ПИа).
